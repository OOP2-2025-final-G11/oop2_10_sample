# アプリ名: LibraryManagementSystem

概要:
このアプリは図書館理システムです。図書の管理を行うことができます。
ユーザの登録、本の貸出、返却、新しい本のリクエストなどの機能があり、実際の図書館に導入できるようなシステムです。
